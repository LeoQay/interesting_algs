Площадь прямоугольников

Дано N прямоугольников со сторонами, параллельными осям координат.
Требуется определить площадь фигуры, образованной объединением данных прямоугольников.

Входные данные
В первой строке находится число прямоугольников - N.
Затем идут N строк, содержащих по 4 числа:
x1, y1, x2, y2 - координаты двух противоположных углов прямоугольника.
1 <= N <= 100,
координаты целые и по абсолютному значению не превосходят 10 000.

Выходные данные
Вывести одно число - площадь фигуры.
